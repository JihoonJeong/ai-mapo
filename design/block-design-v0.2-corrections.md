### 게임 설계 핵심 인사이트

**발견 1: "상업지역 없는 상권"이 마포구의 본질이다**

연남동(상업 0%), 망원1동(상업 0%), 성산1동(상업 0%) — 이 동들에는 각각
경의선숲길 카페거리, 망원시장+망리단길, 월드컵시장이 있다. 전부 **주거지역 위
상업 활동**이다. 서교동도 상업은 8%뿐이고 나머지는 준주거+주거.

→ 게임 메카닉: **"주거지역 상업화"가 젠트리피케이션의 정의 그 자체.**
   구획의 법적 용도(zoning)와 실제 기능(function)이 다를 때 갈등이 발생.

**발견 2: 녹지가 면적의 큰 부분을 차지한다**

합정동 49%, 망원1동 49%, 상암동 82%, 성산2동 30%, 서강동 27%.
대부분 한강 둔치, 월드컵공원, 성미산, 노고산 등.

→ v0.1에서 PARK 구획을 과소 배치했다. 보정 필요.
→ 단, 녹지 면적이 크다고 구획 수를 늘리진 않는다. 녹지는 "조작 가능성"이 적으므로
   1~2개 구획으로 묶되, weight에서 만족도 기여를 높인다.

**발견 3: 준주거 = 전환 지대 = 게임의 레버**

아현동(16%), 서교동(13%), 성산2동(13%), 염리동(10%), 공덕동(8%), 대흥동(8%).
준주거지역은 주거+상업이 모두 허용되는 완충지대.

→ 게임 메카닉: 준주거 구획은 "상업화 촉진" or "주거 보호" 정책 선택지 제공.
   양쪽 정책 모두 효과가 있지만 트레이드오프.

**발견 4: 용강동·대흥동·도화동이 예상보다 상업적**

용강동 17%, 대흥동 15%, 도화동 13% — v0.1에서 이 동들을 "주거 중심"으로 봤는데,
실제로는 마포대로·백범로 대로변을 따라 일반상업지역이 길게 형성.

→ 이 동들의 대로변 구획을 `BIZ_OFFICE` or `COM_LOCAL`로 격상.

## 1. 새로운 구획 속성: `zoning` 필드 추가

### 설계 변경

v0.1의 `type`은 "실제 기능" 기준으로 유지.
새 필드 `zoning`을 추가하여 "법적 용도지역"을 기록.
**type ≠ zoning인 구획이 게임의 갈등 지점.**

```javascript
{
  id: "yeonnamdong_B1",
  name: "경의선숲길 남단",
  dong: "연남동",
  type: "COM_LOCAL",                    // 실제 기능: 카페·음식점 상권
  zoning: "제2종일반주거지역(7층이하)", // 법적 용도: 주거지역
  zoningConflict: true,                 // type ≠ zoning → 갈등 플래그
  traits: ["주거지역_상업화", "젠트리피케이션", "관광"],
  // ...
}
```

### zoningConflict 메카닉

`zoningConflict: true`인 구획에서:
- **주민 민원 확률 증가**: 소음, 쓰레기, 주차 문제
- **임대료 상승 가속**: 상업 수요가 주거 지가를 끌어올림
- **"상업지역 전환" 정책 옵션**: 구청장이 용도지역 변경 추진 가능
  - 비용: 높음 (행정 절차, 주민 반발)
  - 효과: 갈등 해소, but 주거 기능 영구 상실
- **"주거 보호" 정책 옵션**: 영업시간 제한, 업종 제한
  - 비용: 낮음
  - 효과: 주민 만족도↑, but 상권 활력↓

## 4. 새로운 게임 메카닉: 용도지역 갈등 시스템

### 개요

zoningConflict 구획에서 발생하는 갈등을 관리하는 시스템.
기존 수치 설계의 만족도·경제 모델과 연동.

### 갈등 레벨

```
Level 0 (잠재): 주거지역인데 상업 시설 1~2개. 아직 민원 없음.
Level 1 (발현): 상업 시설 3개+. 소음·주차 민원 시작. 만족도 -2/턴.
Level 2 (심화): 임대료 급등. 기존 주민 이탈 시작. 인구 -1%/턴.
Level 3 (위기): 젠트리피케이션 완료 위기. 상권 특색 감소 시작.
```

### 구청장 대응 옵션

```
A. 용도지역 변경 추진 (상업지역으로 전환)
   - 비용: 높음 (50억+), 시간: 8턴
   - 효과: 갈등 해소, 세수 증가
   - 부작용: 주거 기능 영구 상실, 인접 주거 구획 압력 증가

B. 상생 프로그램 (영업시간 제한, 소음 관리, 주차 대책)
   - 비용: 중간 (10억/분기)
   - 효과: 갈등 Level -1, 만족도 회복
   - 부작용: 상권 활력 소폭 감소

C. 주거 보호 강화 (업종 제한, 신규 영업 억제)
   - 비용: 낮음
   - 효과: 갈등 진행 중단
   - 부작용: 상권 위축, 세수 감소

D. 방관 (아무것도 안 함)
   - 비용: 0
   - 효과: 시장 논리대로 진행 (대부분 Level 상승)
```
